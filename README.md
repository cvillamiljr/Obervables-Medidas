# Observables y Medidas 👓🔍

### El proyecto se divide en los siguientes archivos:

- El archivo [Quantum.py](https://github.com/cvillamiljr/Obervables-Medidas/blob/main/Quantum.py) consta de la libreria para simular el primer sistema cuántico descrito en la sección 4.1.

- El archivo [Ejercicios.py](https://github.com/cvillamiljr/Obervables-Medidas/blob/main/Ejercicios.py) que contiene los ejercicios 4.3.1 y 4.3.2 

- El archivo [PruebasObservables.py](https://github.com/cvillamiljr/Obervables-Medidas/blob/main/Pruebas.py) que contiene las pruebas de los ejercicios 4.3.1 y 4.3.2 y la solución de los ejercicios 4.4.1 y 4.4.2 

- Y los archivos necesarios para realizar tanto las operaciones de simulación y los ejercicios propuestos en el capítulo 4:

    + [clasicToQuantum.py](https://github.com/cvillamiljr/Obervables-Medidas/blob/main/clasicToQuantum.py)
    + [ComplexLibrary.py](https://github.com/cvillamiljr/Obervables-Medidas/blob/main/ComplexLibrary.py)

### Recomendaciones👍

Es un proyecto totalmente construido en python, por esta razón va a poder editar y compilar este proyecto con cualquier editor de este lenguaje.
Para poder compilar las simulaciones y los ejercicios debe tener en cuenta que todos los archivos presentes en este repositorio porque todo está importado y enlazado correctamente para que funcione de la mejor manera.
